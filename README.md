# low-code-automation-hw
Python + Playwright Automation Tests

# Project Overview
This repository contains automated tests written in Python using Playwright and pytest. 
It demonstrates UI and API testing scenarios.

# Tests Included
1. E-commerce product workflow  
   - Search, sort and add products to cart. 
   - Verify product title, quantity and price.

2. Form Validation Test 
   - Fill out a sample form and submit.  
   - Verify all input values are displayed correctly.

3. API & UI Contact Management
   - Login via API, add/update/delete contacts.  
   - Verify data consistency between UI and backend.
